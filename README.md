# 🌊 寻找最佳浪点

基于AI智能算法的7天浪况预测与冲浪推荐系统

## 🎯 项目特点

### 核心功能
- **7天详细预测**: 提供未来7天的精确浪况预测
- **AI智能解读**: 基于四因素预测模型的智能分析
- **个性化推荐**: 分别针对长板和短板提供TOP3推荐
- **分区域展示**: 按海南、青岛、舟山地区分类展示

### 技术亮点
- **修正算法**: 基于实际数据分析，大幅优化预测精度
- **地理优化**: 根据不同地理位置调整预测参数
- **响应式设计**: 支持PC和移动端访问
- **纯前端实现**: 无需服务器，直接运行

## 🔬 算法原理

### 四因素预测模型
基于海洋物理学原理，综合考虑：

1. **地形影响**: 海湾形状对浪高的影响
2. **海床深度**: 浅水效应对波浪的衰减
3. **海床类型**: 不同底质对波浪能量的影响
4. **潮汐变化**: 潮汐周期对近岸浪高的调节

### 修正参数
- 基础浪高范围: 0.08-0.33m (相比原0.15-0.65m大幅降低)
- 综合衰减率: 92-98% (解决预测偏高问题)
- 地理位置优化: 根据纬度调整基础参数
- 中国近岸特殊衰减: 额外30%衰减
